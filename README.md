# Simple Interest Calculator

A simple Bash-based calculator that computes simple interest based on user input.

## Project Details

- **Project Name:** Simple Interest Calculator
- **Language:** Bash
- **License:** Apache 2.0

## Description

This project provides a command-line tool that calculates simple interest
given the principal amount, rate of interest, and time period.

## Usage

bash simple-interest.sh

## Formula

Simple Interest = (Principal × Rate × Time) / 100
